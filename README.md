# Python for Bioinformatics

A structured, hands-on journey building practical bioinformatics skills from the ground up — Python, Unix, statistics, and reproducible workflows — applied to real biological data.

## About This Repository

This repo documents my progression from Python fundamentals toward applied bioinformatics. Every concept is tied to real biological data (gene sequences, expression values, sequencing reads), and the work is organized to demonstrate growing competency through practice exercises and mini-projects.

## Repository Structure

python-bioinformatics/
├── 01-basics/          # Python fundamentals (variables, strings, lists, loops, dictionaries)
├── projects/           # Mini-projects applying concepts to biological problems
├── unix-practice/      # Command-line exercises for data processing
├── learning-log/       # Notes, command references, and weekly learning logs
└── learning_journal.md # Weekly progress summary

## Projects

| Project | Description | Concepts Used |
|---------|-------------|---------------|
| DNA Sequence Analyzer | Calculates nucleotide counts and GC content from DNA sequences | Strings, counting, basic arithmetic |
| Gene Expression Tracker | Stores and summarizes gene expression data with formatted reporting | Lists, indexing, `zip()`, loops |
| Codon Frequency Analyzer | Splits DNA into codons and counts frequency of each | Loops, dictionaries, sequence slicing |

## Skills Developed

**Python**
- Core data types, control flow, functions
- List and dictionary operations
- Sequence manipulation (slicing, parsing)

**Unix / Command Line**
- Text processing with grep, sort, uniq, cut
- Building data pipelines with pipes
- Pattern matching for biological file formats

**Statistics**
- P-values and statistical significance
- Multiple testing problem and FDR correction

**Tools & Workflow**
- Git/GitHub version control
- Conda environment management
- Jupyter notebooks / VS Code
